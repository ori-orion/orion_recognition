# orion_recognation

## Launch
```
$ sim_mode
<local>~$ source catkin_ws/devel/setup.bash
<local>~$ roslaunch orion_recognition recognition.launch
```

## To run check for object action server
```
$ sim_mode
<local>~$ source catkin_ws/devel/setup.bash
<local>~$ rosrun orion_recognition check_for_object_server_node.py
```
